What is Bash? Why install Bash? Enabling developer features Enable & Install Linux
Bash is a unix shell used in linux and the GNU operating system. "Bash” is an abbreviation for the ‘Bourne-Again SHell’,a pun on the name of the "Bourne Shell" it replaces and being "born again". Bash is the default of interactive shell on various distributions of linux due to its popularity on Linux. With the release of Ubuntu on Windows, it is now possible to install bash on Windows as well.
The answer is the open-source development community. Powershell(Windows) and Bash(Linux) have different focus and design. Due to increase in the open-source programming languages, and the developer community adopting them, it becomes essential for an operating system to support it. Having the Bash shell available to download in Windows removes the extra steps that the developers have to follow to use the same programming languages on Windows. Compared to using an operating system that has native support for bash, using powershell means more effort and a steeper learning curve.

Now let's get to the installation

First step is to enable the developer feature of windows. Simply go to

Settings > Updates & Security > Use developer features

and select developer mode option. The device will then restart.

Go to

Control panel > Turn windows feature on or off > Check "windows subsystem for linux"

After restarting go to

windows store > download "Ubuntu".

After installing, type "bash" in command prompt and bash is ready to be used.
